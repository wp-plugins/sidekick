== Description ==

The Sidekick Plugin provides real-time, guided WordPress help right from your dashboard, whenever you need it. It’s not a video, it’s not an overlay. It’s real help happening in real-time solving real problems and questions.

With the click of a button, you’ll get a step-by-step answer to your question instantly. So you can stop waiting for your developer call you back.

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and upload the Sidekick plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a WPU logo appear in the bottom right corner of your Dashboard.
